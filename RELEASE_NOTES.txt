DETAILED CHANGELOG (for reference):
===============================================================================

BUG FIXES:
----------
• Fixed crash when opening Settings screen on Android 15 devices
• Resolved status bar overlay issue where content appeared under system bars
• Fixed ActionBar compatibility issues with modern Android versions

NEW FEATURES:
-------------
• Added "About" section in Settings showing:
  - App version and build code
  - Build date and time
  - Copyright information
  - Clickable link to GitHub project for bug reports and source code

DEVICE COMPATIBILITY:
---------------------
• App now exclusively targets Android phones (handheld devices only)
• Removed from Play Store for:
  - Tablets (large/xlarge screens)
  - Chromebooks (Chrome OS devices)
  - Google XR devices (AR/VR headsets)
  - Watches, TVs, and Automotive displays

TECHNICAL IMPROVEMENTS:
-----------------------
• Upgraded Android Gradle Plugin from 8.2.2 to 8.7.3
• Upgraded Gradle wrapper from 8.4 to 8.9
• Full Android 15 (API level 35) support
• Enabled BuildConfig for better version tracking
• Improved app stability on latest Android versions
